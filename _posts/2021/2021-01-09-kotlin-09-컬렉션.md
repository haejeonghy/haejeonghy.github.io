### 필기

### 컬렉션의 구조와 기본

- 컬렉션 : 자주 사용하는 기초적인 자료구조를 모아놓은 일종의 프레임워크
- 코틀린 컬렉션 : 자바의 컬렉션 구조 확장 구현 - 되도록 읽기 전용을 권장
    - List : listOf (읽기 전용) : mutableListOf, arrayListOf (가변형)
    - Set : setOf (읽기 전용) : mutableSetOf, hashSetOf, linkedSetOf, sortedSetOf (가변형)
    - Map : mapOf (읽기 전용) : mutableMapOf, hashMapOf, linkedMapOf, sortedMapOf (가변형)
- 컬렉션 인터페이스 멤버
    - size : 컬렉션의 크기
    - isEmpty() : 컬렉션이 비어있으면 true 반환
    - contains(element: E) : 특정 요소가 있다면 true 반환
    - containsAll(elements: Collection<E>) : 인자로 받아들인 컬렉션이 있다면 true 반환
- MutableCollection 인터페이스 멤버 메서드
    - add(element: E) : 인자로 전달받은 요소를 추가하고 true를 반환, 이미 요소가 있거나 중복 허용되지 않으면 false 반환
    - remove(element: E) : 인자로 전달받은 요소를 삭제하고 true 반환, 삭제하려는 요소가 없다면 false 반환
    - addAll(elements: Collection<E>) : 컬렉션을 인자로 전달 받아 모든 요소를 추가하고  true를 반환하며 실패하면 false 반환
    - removeAll(elements: Collection<E>) : 컬렉션을 인자로 전달 받아 모든 요소를 삭제하고  true를 반환하며 실패하면 false 반환
    - retainAll(elements: Collections<E>) : 인자로 전달받은 컬렉션의 요소만 보유한다. true를 반환하며 실패하면 false 반환
    - clear() : 컬렉션의 모든 요소 삭제

### List

- List : 순서에 따라 정렬된 요소를 가지는 컬렉션
- 불변형 List 생성하기 : listOf

```kotlin
public fun<T> listOf(vararg elemnts: T): List<T>

// 생성 예시
var numbers: List<Int> = listOf(1, 2, 3, 4, 5)

// 자료형 여러개 포함 가능
val mixedTypes = listOf("Hello", 1, 2, 3, 4, 5)

// 인덱스 지정하여 반복 가능
for (index in fruits.indices) {
}
```

- emptyList() : 비어있는 리스트 생성 가능

```kotlin
val emptyList: List<String> = emptyList<String>()
```

- listOfNotNull() : null을 제외한 요소만 반환해 List 구성 가능

```kotlin
val nonNullsList: List<Int> = listOfNotNull(2, 45, null, 5, null)
println(nonNullsList) // 2, 45, 2, 5
```

- 주요 메서드
    - lastIndexOf(element: E) : 인자로 받은 요소가 마지막으로 나타나는 인덱스 반환 없으면 -1
    - indexOf(element: E) : 인자로 받은 요소가 첫 번째로 나타나는 인덱스를 반환하며 없으면 -1
    - listIterator() : 목록에 있는 iterator 반환
    - subList(fromIndex: Int, toIndex: Int) : 특정 인덱스의 from과 to 범위에 있는 요소 목록 반환
- 가변형 List 생성 : arrayListOf()

```kotlin
public fun <T> arrayListOf(vararg elements: T): ArrayList<T>

val stringList: ArrayList<String> = arrayListOf<String>("Hello", "Kotlin", "Wow")
stringList.add("Java") // 추가 가능
stringList.remove("Hello")  // 삭제 가능
```

- 가변형 mutableListOf() 함수

```kotlin
public fun <T> mutableListOf(vararg elements: T): MutableList<T>

val mutableList: MutableList<String> = mutableListOf<String>("Kildong", "Dooly", "Chelsu")
mutableList.add("ben")
mutableList.removeAt(1) // 인덱스 1번 삭제 가능
mutableList[0] = "Sean" // 인덱스 0번을 변경 set과 같은 역할 

// 불변형 리스트 -> 가변형 변경 toMutableList()

val names: List<String> = listOf("one", "two", "three")
val mutableNames = names.toMutableList() // 새로운 가변형 List 생성
```

### Set과 Map 활용

- Set : 정해진 순서가 없는 요소들의 집합 = 모든 요소가 유일해야 함
- Set 생성 : setOf() : 읽기 전용인 불변형 Set<T> 자료형 반환

```kotlin
val mixedTypesSet = setOf("Hello", 5, "World", 'c') // 자료형 혼합 초기화
val intSet: Set<Int> = setOf<Int>(1, 5, 5) 정수형만 초기화 
```

- 가변형 mutableSetOf() 함수 : 요소의 추가 및 삭제 가능한 집합

```kotlin
val animals = mutableSetOf("Lion", "Dog", "Cat")
animals.add("Dog") // 기존에 저장되어있는 요소로 변화 없음
animals.remove("Lion") // Lion 요소 삭제 
```

- hashSetOf() 함수 : 내부적으로 키와 인덱스를 이용해 검색과 변경 등을 매우 빠르게 처리할 수 있는 자료구조

```kotlin
val intsHashSet: HashSet<Int> = hashSetOf(6, 3, 4, 7)
intsHashSet.add(5) // 추가
intsHashSet.remove(6) // 삭제
```

- sortedSetOf() : TreeSet 컬렉션을 정렬된 상태로 반환함 - 최악으로 요소 배치가 되어도 검색 등의 처리에서 일정한 시간을 보장하는 자료구조 : 검색과 정렬이 뛰어남

```kotlin
val intsSortedSet: TreeSet<Int> = sortedSetOf(4, 1, 7, 2) 
intsSortedSet.add(6) // 
intsSortedSet.remove(1) 
```

- linkedSetOf() : 자바의 LinkedHashSet 자료형 반환 함수 : 메모리 저장 공간을 좀 더 효율적으로 사용할 수 있음

```kotlin
val intsLinkedHashSet: java.util.LinkedHashSet<Int> = linkedSetOf(35, 21, 76, 26, 75) 
intsLinkedHashSet.add(4)
intsLinkedHashSet.remove(21)
intsLinkedHashSet.clear()
```

- Map : 요소가 키와 값 쌍 형태로 저장됨 = 키는 중복 불가, 값은 중복 가능
- 불변형 mapOf() 함수

```kotlin
val map: Map<키 자료형, 값 자료형> = mapOf(키 to 값[, ...])

val langMap: Map<Int, String> = mapOf(11 to "java", 22 to "kotlin", 33 to "C++")
// 맵 반복
for ((key, value) in langMap) {
}
```

- Map 함수
    - size : Map 컬렉션 크기 반환
    - keys : Map의 모든 키 반환
    - values : Map의 모든 값 반환
    - isEmpty() : Map이 비어있는지 확인하고 비어있으면 true, 아니면 false 반환
    - containsKey(key: K) : 인자에 해당하는 키가 있다면 true, 아니면 false 반환
    - containsValue(value: V) : 인자에 해당하는 값이 있다면 true, 아니면 false 반환
    - get(key: K) : 키에 해당하는 값 반환, 없으면 null 반환
- 가변형 mutableMapOf() 함수 : 추가, 삭제가 가능한 가변형 Map 정의

```kotlin
val capitalCityMap: MutableMap<String, String> = 
	mutableMapOf("Korea" to "seoul", "china" to "beijing", "Japan" to "tokyo")

capitalCityMap.put("UK", "London") // 요소의 추가
capitalCityMap.remove("China") // 요소의 삭제
```

- MutableMap 함수
    - putAll(from: Map<out K, V>) : 인자로 주어진 Map 데이터를 갱신하거나 추가
    - clear() : 모든 요소 삭제
- HashMap, SortedMap, LinkedHashMap 사용 가능

### 컬렉션의 확장 함수

- 컬렉션의 연산 : 연산자 사용하여 컬렉션에 대해 더하거나 빼는 기능 수행 가능

```kotlin
List<Int> - listOf(3, 4, 5) 
mapOf("hi" to 1, "hello" to 2, "goodbye" to 3) + Pair ("Bye", 4)

...
```

- 요소의 순환
    - forEach // 각 요소를 람다식으로 처리
    - forEachIndexed // forEach에 인덱스 포함

```kotlin
list.forEach { print("$it")} // 컬렉션 반환하지 않음
list.forEachIndexed { index, value -> println("index[$index]: $value")} // 인덱스 포함
list.onEach { print(it)} // 각 요소를 람다식으로 처리한 후 컬렉션으로 반환
```

- 요소의 개수 반환 : count

```kotlin
list.count {it % 2 == 0}) // 조건에 맞는 요소 개수 반환
```

- 최댓값과 최솟값의 요소 반환

```kotlin
list.max() // 리스트의 최댓값
list.min() // 리스트의 최솟값

map.maxBy { it.key} // 키를 기준으로 최댓값
map.minBy { it.key} // 키를 기준으로 최솟값
```

- 각 요소에 정해진 식 적용하기
    - fold : 초기값과 정해진 식에 따라 처음 요소부터 끝 요소에 적용해 값 반환
    - reduce : fold와 동일하나 초기값 사용 안함

```kotlin
list.ford(4) { total, next -> total + next}
// 마지막 요소에서 처음 요소로 반대로 적용
list.fordRight(4) { total, next -> total + next } 
list.reduce { total, next -> total + next } // fold와 동일하지만 초기값 사용 안함 
```

- 모든 요소 합산 : sumBy

```kotlin
listPair.sumBy { it.second} 
```

- 요소의 일치 여부 검사

```kotlin
list.all { it < 10} // 모든 요소가 조건에 부합해야 true 반환 
list.any { it % 2 == 0 } // 최소한 하나 혹은 그 이상의 특정 요소가 일치해야 true 반환

list.contains(2) // 요소가 포함되어있으면 true 반환
2  in list // 요소가 포함되어있으면 true 반환
list.containsAll(listOf(1, 2, 3)) // 모든 요소가 포함되어 있으면 true 반환

list.none() // 요소가 없으면 true 반환
list.none { it > 6 } // 6이상 이 없을경우 true 반환

list.isNotEmpty() // 컬렉션이 비어있지 않은지 검사 

list.filter { it % 2 == 0 } // 필터 식에 따라 요소 골라내기
list.filterNot { it % 2 == 0 } // 식 이외의 요소 골라내기
list.filterNotNull() // null을 제외

list.filterIndexed { idx, value -> idx != 1 && value % 2 == 0} // 인덱스와 함께 추출
// 추출 후 가변형 컬렉션으로 변환
list.filterIndexedTo(mutableListOf()) { idx, value -> idx != 1 && value % 2 == 0 }

map.filterKeys { it != 11} // 키 11을 제외한 요소
map.filterValues { it == "Java" } // 값이 java인 요소 

list.filterIsInstance<String>() // 여러 자료형 요소 중 원하는 자료형만 골라냄
```

- 특정 범위 잘라내거나 반환하기

```kotlin
list.slice(listOf(0, 1, 2)) // 특정 인덱스 요소들을 잘라서 반환하기

list.take(2) // 앞 두 요소 반환
list.takeLast(2) // 마지막 두 요소 반환
list.takewhile { it < 3} // 조건식에 따른 반환

list.drop(3) // 처음부터 n개의 요소를 제외한 list 반환
list.dropWhile{ it < 3} // 3 미만을 제외하고 반환
list.dropLastWhile { it > 3} // 3 초과를 제외하고 반환
```

- 각 요소의 반환

```kotlin
list.component1() // 첫 번째 요소인 1 반환
```

- 합집합과 교집합

```kotlin
listRepeated.distinct() // 중복 요소는 하나로 취급해 List 반환
list.intersect(listOf(5, 6, 7, 8)) // 교집합 요소만 골라냄
```

- 요소의 매핑

```kotlin
list.map { it * 2} // 컬렉션에 주어진 식을 적용해 새로운 컬렉션 반환
// 컬렉션에 인덱스를 포함하고 주어진 식을 적용해 새로운 컬렉션 반환
list.mapIndexed { index, it -> index * it} 
// null을 제외하고 식을 적용해 새로운 컬렉션 반환
list.mapNotNull { it?.times(2)} 
// 각 요소에 식을 적용한 후 이것을 하나로 합쳐 새로운 컬렉션 반환
list.flatMap { listOf(it, 'A')}
// 주어진 식에 따라 요소를 그룹화하고 이것을 다시 Map으로 반환
list.groupBy { if (it % 2 == 0) "even" else "odd" }
```

- 요소 처리와 검색

```kotlin
list.elementAt(1) // 인덱스에 해당하는 요소 반환
// 인덱스를 벗어나는 경우 식에 따라 결과 반환
list.elementAtOrLese(10, { 2 * it})
// 인덱스를 벗어나는 경우 null 반환
list.elementAtOrNull(10)

// 식에 일치하는 첫 요소 반환
list.first { it.second == 200 }
// 식에 일치하는 마지막 요소 반환
list.last { it.second == 200}
// 식에 일치하지 않는 경우 null 반환
list.firstOrNull { it.first == "E"}
list.lastOrNull {it.first == "E"}

//주어진 요소에 일치하는 첫 인덱스 반환
list.indexOf(4) 
// 람다식에 일치하는 첫 요소의 인덱스 반환, 없으면 -1
list.indexOfFirst { it % 2 == 0} 
// 주어진 요소에 일치하는 가장 마지막 인덱스 반환
list.lastIndexOf(5) 
// 람다식에 일치하는 마지막 요소의 인덱스 반환, 없으면 -1
list.indexOfLast{it % 2 == 0}

// 람다식에 일치하는 요소 하나 반환
list.single { it.second == 100}
list.singleOrNull { it.second == 500}

// 요소에 대해 이진 탐색 후 인덱스 변환
list.binarySearch(3)
// 조건식을 만족하는 첫 번째 검색된 요소 반환, 없으면 null
list.find { it > 3}
```

- 컬렉션의 분리와 병합

```kotlin
list.union(list2) // 중복 요소 포함하지 않고 두 리스트 합침
list.plus(list2) // 중복 요소 포함 합침 

// 주어진 식에 따라 2개의 컬렉션으로 분리해 Pair로 변환
var part = list1.partition { it % 2 == 0 } 

// 컬렉션에서 동일한 인덱스끼리 Pair를 만들어 반환
val zip = list.zip(listOf(7, 8)) 
```

- 순서와 정렬

```kotlin
// 뒤집힌 순서로 컬렉션 반환
list.reversed()
// 요소를 정렬한 후 정렬된 컬렉션 반환
list.sorted()
// 내림차순 정렬 
list.sortedDescending()
// 특정 비교식에 의해 정렬된 컬렉션 반환
list.sortedBy { it % 3 }
list.sortedByDescending { it % 3} 
```

### 시퀀스

- 순차적인 컬렉션으로 요소의 크기를 특정하지 않고 나중에 결정할 수 있는 특수한 컬렉션
- 요소 값 생성 : generateSequence()

```kotlin
val nums: Sequence<Int> = generateSequence(1) { it + 1} 
// 원하는 요소 개수만큼 획득하고 toList를 통해 list 컬렉션으로 반환
nums.take(10).toList()
// map, filter 가능
```

- 요소 값 가져오기

```kotlin
// 중간 연산 결과 없이 한 번에 끝까지 연산한 후 결과를 반환할때 asSequence 사용
// 병렬 처리 되어 처리 성능이 좋아짐 
val listSeq = list1.asSequence()
	.map {}
	.filter {}
	.toList()
```
