### 필기

### 조건문

- in : 변수 이름 in 시작값..마지막값 (시작값과 마지막값 포함)

```kotlin
if(socre in 80.0..89.9) { // 80.0 이상 89.9 이하
}
```

- when : 인자 구분, 등급에 따라 표현
- is : 특정 자료형 검사

```kotlin
val str = "안녕하세요."
val result = when(str) {
	is String -> "문자열입니다."
	else -> false
}
```

### 반복문

- for 범위문 :

```kotlin
for ( 요소 변수 in 컬렉션 또는 범위) { 반복할 본문}

for (x in 1..5){
	print(x)
}

for (i in 5 downTo 1){ // 5부터 1까지 역순
}

for (i in 1..5 step 2){ // 1부터 5까지 2씩 증가
}

for (i in 5 downTo 1 step 2){ // 5부터 1까지 2씩 감소
}
```

- do ~ while : do 본문 무조건 한번 실행 후 while 조건에 해당할 동안 반복
    
    ```kotlin
    do {
    
    	본문
    
    } while (조건식)
    ```
    

### 흐름의 중단과 반환

- 람다식에서 return 사용하기
    - inline lambda의 경우 사용가능

```kotlin
inline fun inlineLambda(a: Int, b: Int, out: (Int, Int) -> Unit) {
	out(a, b)
}

fun retFunc() {
	inlineLambda(13, 3) { a, b ->
		val result = a + b
		if(result > 10) return
	}
}
```

- 라벨과 함께 return 사용하기

```kotlin
람다식 함수 이름 라벨 이름@ {
	return@라벨 이름
}

fun inlineLambda(a: Int, b: Int, out: (Int, Int) -> Unit) {
	out(a, b)
}

fun retFunc() {
	inlineLambda(13, 3) lit@{ a, b -> // 람다식블럭 시작 부분에 라벨 지정
		val result = a + b
		if(result > 10) return@lit // 라벨을 사용한 블록의 끝부분으로 반환
	}
}
```

- 암묵적 라벨 : 람다식 명칭을 그대로 라벨처럼 사용함

```kotlin
fun retFunc() {
	inlineLambda(13, 3) { a, b ->
		if(result > 10) return@inlineLambda // 람다식 이름을 라벨처럼 사용하고 return함
	}
}
```

- 익명 함수를 사용한 반환

```kotlin
fun retFunc() {
	inlineLambda(13, 3, fun(a, b) {
		val result = a + b
		if(result > 10) return
	})
}

//라벨을 사용하려면
val getMessage = lambda@ { num: Int -> 
	if(num !in 1..100){
		return@lambda "Error"
	}
	"Success"
}
```

- break; : 해당 키워드를 사용한 지점에서  for이나 while, do~while문 루프를 빠져나오게 됨

```kotlin
fun labelBreak() {
	for (i in 1..5) {
		second@ for (j in 1..5) { // break과 라벨 함께 사용하여 반복문 중단 위치 변경
			if( j == 3) break // 두번째 for문 탈출
		}
}

fun labelBreak() {
	first@ for (i in 1..5) {
		second@ for (j in 1..5) { 
			if( j == 3) break@first // 첫번째 for문 탈출
		}
}
```

- continue : 이후 본문을 계속 진행하지 않고 다시 반복 조건을 살펴보게 됨
