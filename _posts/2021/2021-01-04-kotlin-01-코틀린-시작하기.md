### 키워드

- 용도
- 장점
- main()

### 필기

### 멀티 플랫폼

1. Kotlin/JVM : 자바 가상 머신에서 동작하는 앱
    - 자바 앱, 안드로이드 앱
2. Kotlin/JS : 자바 스크립트로 웹 브라우저에서 동작하는 앱
    - DB, 서버, 클라이언트
3. Kotlin/Native : [LLVM](https://ko.wikipedia.org/wiki/LLVM) 컴파일러 이용 여러 플랫폼 타깃으로 하는 앱
    - 안드로이드, iOS

### 장점

- 자료형 오류 미리 잡을 수 있는 정적 언어 : 컴파일 될 때 자료형을 검사하여 확정하므로, 자료형 오류를 초기에 발견할 수 있다.
- 널 포인터 익셉션으로 인한 프로그램 중단 예방
- 간결하고 효율적
- 함수형 프로그래밍, 객체 지향 프로그래밍 모두 가능
- 세미콜론 생략 가능
- 함수형 프로그래밍 + 객체 지향 프로그래밍 모두 지원
    - 다중 패러다임 언어

### main()

- 작성된 코틀린 코드는 JVM에서 실행되며, main() 함수가 있는 파일 이름 기준으로 자바 클래스 자동 생성됨 : 클래스를 선언하지 않고 main() 함수만 실행해도 컴파일됨
