# cmd 단축키
Created: 2025년 2월 16일 오후 4:20
보관소: No
최종 편집 일시: 2026년 3월 14일 오전 1:08

watch: 반복해서 같은 명령 실행

- ctrl + a : 커맨드 맨 앞으로 이동
- ctrl + e : 커맨드 맨 뒤로 이동
- ctrl + r : 커맨드 히스토리 검색
1. `Cmd + N`: 새 터미널 창을 엽니다.
2. `Cmd + T`: 새 탭을 엽니다.
3. `Cmd + W`: 현재 탭을 닫습니다.
4. `Cmd + Q`: 터미널을 완전히 닫습니다.
5. `Cmd + 클릭`: 링크를 열거나 파일 경로를 Finder에서 보여줍니다.
6. `Cmd + Shift + K`: 새로운 SSH 세션을 시작합니다.
7. `Cmd + Shift + H`: 홈 디렉토리로 이동합니다.
8. `Cmd + Shift + D`: 데스크탑 디렉토리로 이동합니다.
9. `Cmd + Shift + A`: 응용 프로그램 디렉토리로 이동합니다.
10. `Cmd + Shift + O`: 문서 디렉토리로 이동합니다.
11. `Cmd + Shift + U`: 유틸리티 디렉토리로 이동합니다.
12. `Cmd + Shift + V`: 클립보드의 내용을 붙여넣습니다.
13. `Cmd + Shift + M`: 이메일 주소를 복사합니다.
14. `Cmd + Shift + S`: 터미널 설정을 엽니다.
15. `Cmd + ]` / `Cmd + [`: 탭 간에 이동합니다.
16. `Ctrl + A`: 현재 라인의 시작으로 커서를 이동합니다.
17. `Ctrl + E`: 현재 라인의 끝으로 커서를 이동합니다.
18. `Ctrl + U`: 커서 위치부터 라인 시작까지의 텍스트를 지웁니다.
19. `Ctrl + K`: 커서 위치부터 라인 끝까지의 텍스트를 지웁니다.
20. `Ctrl + W`: 커서 앞의 단어를 삭제합니다.
21. `Ctrl + L`: 터미널 화면을 클리어합니다.
22. `Ctrl + C`: 현재 실행 중인 명령을 중단합니다.
23. `Ctrl + D`: 현재 쉘을 종료합니다. 입력이 없을 때 `EOF` (End of File)을 나타냅니다.
24. `Ctrl + Z`: 현재 실행 중인 프로세스를 일시 중지하고 백그라운드로 보냅니다.
25. `Ctrl + R`: 이전에 사용했던 명령을 검색합니다.
26. `Ctrl + P`: 이전 명령어로 이동합니다 (위 방향키와 유사).
27. `Ctrl + N`: 다음 명령어로 이동합니다 (아래 방향키와 유사).
28. `Ctrl + Y`: 'kill ring'에 저장된 텍스트를 현재 커서 위치에 붙여넣습니다.
29. `Ctrl + T`: 커서 앞의 두 문자를 서로 바꿉니다.
30. `Alt + B`: 커서를 현재 위치에서 이전 단어의 시작 부분으로 이동합니다.
31. `Alt + F`: 커서를 현재 위치에서 다음 단어의 끝 부분으로 이동합니다.
32. `Alt + D`: 커서 위치에서부터 단어의 끝까지 텍스트를 삭제합니다.
33. `Alt + Backspace`: 커서 위치에서부터 이전 단어의 시작까지 텍스트를 삭제합니다.
34. `Alt + T`: 커서 앞의 두 단어를 서로 바꿉니다.
35. `Alt + U`: 커서 위치에서 단어의 끝까지 대문자로 변경합니다.
36. `Alt + L`: 커서 위치에서 단어의 끝까지 소문자로 변경합니다.
37. `Alt + C`: 커서 위치에서 단어의 첫 글자를 대문자로 변경하고 나머지는 소문자로
