### **📝 Golang에서의 rune 정리**

---

### **1. rune의 기본 개념**

- *rune**은 **유니코드 코드 포인트 (Unicode Code Point)**를 표현하는 자료형
- *int32**의 별칭 (32비트 정수)
- 개별 문자(char)를 나타내는 기본 단위

---

### **2. rune의 주요 특징**

- **단일 문자**를 표현
- 유니코드 범위: **0** ~ **1,114,111 (0x10FFFF)**
- 기본적으로 **UTF-8**로 인코딩된 문자열에서 개별 문자 단위로 처리
- *바이트(byte)**와는 다르게 **문자 단위**로 정확한 길이 계산 가능

---

### **3. 문자열과의 차이점**

- **string**: 바이트 슬라이스 ([]byte)
- **rune**: 문자 슬라이스 ([]rune)

---

### **4. 사용 예시**

```go
package main

import "fmt"

func main() {
    // 문자열에서 개별 문자 추출
    str := "안녕하세요, Go!"
    runes := []rune(str)

    fmt.Println("runes:", runes)         // [50504 45397 54616 49464 50836 44 32 71 111 33]
    fmt.Printf("첫 번째 문자: %c\n", runes[0]) // 안
    fmt.Printf("두 번째 문자: %c\n", runes[1]) // 녕
    fmt.Printf("세 번째 문자: %c\n", runes[2]) // 하

    // 문자열 길이 비교
    fmt.Println("len (byte):", len(str))       // 19 (바이트 단위)
    fmt.Println("len (rune):", len(runes))     // 10 (문자 단위)

    // 문자열 자르기 (문자 단위)
    fmt.Println(string(runes[:5])) // 안녕하세요
}
```

---

### **5. 실무에서의 활용 사례**

- 문자열 자르기 (문자 단위)
- 문자열 길이 계산 (정확한 문자 수)
- 문자열 역순 처리 (Palindrome 검사)
- 이모지나 특수 문자 필터링

---

**👉 정리:**

- *rune**은 개별 문자를 처리할 때 유용하며, 다국어 지원이 중요한 프로그램에서 필수적인 개념입니다.
- 기본적으로 **문자 단위**로 데이터를 처리하고, **UTF-8**의 가변 길이 인코딩 문제를 해결하는 데 핵심적인 역할을 합니다. 💡
