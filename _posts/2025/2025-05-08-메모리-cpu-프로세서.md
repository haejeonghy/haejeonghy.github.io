# 메모리, CPU, 프로세서

### **📝 메모리, CPU, 프로세서 정리**

---

### **1. 메모리 (Memory, RAM)**

- **데이터를 임시로 저장**하는 공간
- CPU가 연산할 때 필요한 데이터를 **빠르게 읽고 쓰기** 위해 사용
- **전원이 꺼지면 데이터가 사라짐 (휘발성)**
- 보통 **GB (기가바이트)** 단위로 크기를 표현
- **DDR4, DDR5**와 같은 세대 차이가 있음

---

### **2. CPU (Central Processing Unit, 중앙 처리 장치)**

- 컴퓨터의 **두뇌** 역할
- 모든 **연산**과 **명령 처리**를 담당
- 기본적인 **산술, 논리, 제어** 연산을 수행
- *프로세서 (Processor)**의 일종
- *코어 (Core)**가 여러 개인 **멀티코어** CPU도 있음

---

### **3. 프로세서 (Processor)**

- **명령어를 실행**하는 장치의 총칭
- CPU도 프로세서의 일종이지만, **GPU**, **DSP**와 같은 다른 유형의 프로세서도 존재
- CPU는 일반적인 **연산**을 담당하고, GPU는 **그래픽 연산**에 특화됨

---

### **4. 요약**

| **구분** | **CPU (Central Processing Unit)** | **메모리 (RAM)** | **프로세서 (Processor)** |
| --- | --- | --- | --- |
| **기능** | 명령어 처리, 연산 | 데이터 임시 저장 | 명령어 실행 |
| **데이터** | 계산된 결과 | 실행 중인 프로그램 데이터 | 명령어 집합 |
| **속도** | 매우 빠름 | 빠름 | 다양함 (CPU, GPU, DSP) |
| **전력 소모** | 높음 | 중간 | 다양함 |
| **휘발성** | 비휘발성 | 휘발성 | 비휘발성 |

---

**👉 정리:**

- **메모리**는 데이터를 **임시로 저장**하는 공간
- **CPU**는 **연산**과 **명령 처리**를 담당하는 컴퓨터의 **두뇌**
- **프로세서**는 **명령어 실행**을 담당하는 모든 장치의 총칭
