# 🛠️ Copybara - 리포지토리 간 코드 변환 및 이동 도구

📅 **최초 작성일**: 2024-05-09

🔗 [GitHub Repository](https://github.com/google/copybara)

---

## 📌 Copybara란?

**Copybara**는 Google에서 내부적으로 사용되는 도구로, **리포지토리 간 코드 변환 및 이동**을 수행하는 기능을 제공합니다.

소스 코드가 여러 저장소에 존재해야 하는 경우, **Copybara를 활용하면 저장소 간 코드 동기화 및 변환을 자동화**할 수 있습니다.

특히, **기밀 저장소 ↔ 공용 저장소 간 코드 유지 관리**에 유용하게 사용됩니다.

---

## 🚀 Copybara의 주요 기능

✅ **리포지토리 간 코드 이동 및 변환**

✅ **기밀 저장소와 공개 저장소 동기화**

✅ **신뢰할 수 없는 저장소의 변경 사항을 검토 후 반영**

✅ **상태 비저장 방식으로 여러 사용자/서비스가 동일한 결과 생성 가능**

---

## 📂 Copybara의 사용 예시

📌 **기밀 저장소 → 공개 저장소**

- 기밀 저장소에서 특정 코드 섹션을 공개 저장소로 전환하여 공유

📌 **공개 저장소 → 기밀 저장소**

- 오픈소스 프로젝트의 변경 사항을 기밀 저장소로 가져와 내부 코드에 적용

📌 **신뢰할 수 없는 저장소 → 신뢰할 수 있는 저장소**

- 공개 저장소의 외부 기여를 검토 후, 내부 신뢰 저장소로 변환 및 반영

📌 **새로운 저장소로 코드 이동**

- 기존 프로젝트의 일부 또는 전체 코드를 새로운 저장소로 마이그레이션

---

## 🔧 Copybara의 원칙

🔹 **하나의 진실 소스(Single Source of Truth)**

- Copybara를 사용하면 **항상 신뢰할 수 있는 저장소**를 하나로 설정해야 합니다.
- 그러나, 여러 저장소에서 기여 및 릴리스가 가능하도록 설계되어 있습니다.

🔹 **상태 비저장 (Stateless)**

- Copybara는 **대상 저장소에만 상태를 저장** (예: 커밋 메시지의 레이블)
- 이를 통해 여러 사용자 및 서비스가 동일한 구성에서 동일한 결과를 얻을 수 있음
