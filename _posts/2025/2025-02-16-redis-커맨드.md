# redis 커맨드
Created: 2025년 2월 16일 오후 4:22
Tags: redis
보관소: No
최종 편집 일시: 2026년 3월 14일 오전 1:08

- flushall - redis key, value 모두 날리기 - 전체 DB 삭제
1. `SET key value`: 주어진 키(key)에 값(value)을 할당합니다.
2. `GET key`: 주어진 키의 값을 조회합니다.
3. `DEL key`: 주어진 키를 삭제합니다.
4. `KEYS pattern`: 패턴에 맞는 모든 키를 나열합니다.
5. `EXISTS key`: 주어진 키가 존재하는지 확인합니다.
6. `EXPIRE key seconds`: 주어진 키의 만료 시간을 설정합니다.
7. `TTL key`: 주어진 키의 남은 만료 시간을 초 단위로 확인합니다.
8. `LPUSH key value`: 리스트의 왼쪽(앞)에 요소를 추가합니다.
9. `RPUSH key value`: 리스트의 오른쪽(끝)에 요소를 추가합니다.
10. `LPOP key`: 리스트의 왼쪽(앞)에서 요소를 제거하고 반환합니다.
11. `RPOP key`: 리스트의 오른쪽(끝)에서 요소를 제거하고 반환합니다.
12. `SADD key member`: 셋에 멤버를 추가합니다.
13. `SMEMBERS key`: 셋의 모든 멤버를 조회합니다.
14. `ZADD key score member`: 정렬된 셋에 멤버와 점수를 추가합니다.
15. `ZRANGE key start stop`: 정렬된 셋에서 범위에 해당하는 멤버들을 조회합니다.
16. flushdb 특정 DB만 날리기
