### **📝 Golang 배열과 슬라이스 차이 요약**

---

### **1. 배열 (Array)**

- **고정 크기**: 선언 시 크기 지정 필수
- **값 타입**: 복사 전달
- **기본값**: int는 0, string은 "", bool은 false
- **메모리 효율적**
- **선언 예시:**

```
var arr [3]int      // [0 0 0]
arr2 := [3]int{1, 2, 3}  // [1 2 3]
```

---

### **2. 슬라이스 (Slice)**

- **가변 크기**: 동적 확장 가능
- **참조 타입**: 참조 전달
- **nil 가능**
- **길이(len)와 용량(cap) 분리**
- **선언 예시:**

```
var slice []int         // nil
slice2 := []int{1, 2, 3} // [1 2 3]
```

---

### **3. 주요 차이점**

| **특징** | **배열 (Array)** | **슬라이스 (Slice)** |
| --- | --- | --- |
| **크기** | 고정 | 가변 |
| **초기화** | 기본값 | nil 가능 |
| **타입** | 값 타입 | 참조 타입 |
| **길이와 용량** | 동일 | 분리 가능 |
| **복사 방식** | 값 복사 | 참조 전달 |

---

**👉 정리:**

배열은 **고정 크기**로 사용하고, 슬라이스는 **가변 크기**로 사용한다는 점이 가장 큰 차이입니다. 배열은 주로 메모리 효율이 중요한 경우, 슬라이스는 데이터의 추가, 삭제가 자주 발생하는 경우 사용합니다. 💡
