# intellij 단축키
Created: 2025년 2월 16일 오후 4:21
보관소: No
최종 편집 일시: 2026년 3월 14일 오전 1:08

goland intellij에서 세로 선택

shift command 8 치고 + shift 누른 채로 아래로 방향키

- alt + enter : parameter 줄 바꿈
- ctrl + l : 라인 넘버 이동 window 키
- ctrl cmd g : 선택한 단어 화면에서 일괄 선택
- ctrl g : 선택한 단어 하나씩 선택
1. `Cmd + N`: 현재 위치에서 새 파일, 클래스, 패키지 등을 생성합니다.
2. `Cmd + O`: 클래스를 검색하여 엽니다.
3. `Cmd + Shift + O`: 파일을 검색하여 엽니다.
4. `Cmd + Alt + L`: 코드 스타일에 맞게 현재 파일을 포맷팅합니다.
5. `Cmd + Shift + A`: 액션 또는 옵션을 검색합니다.
6. `Cmd + B` 또는 `Cmd + 클릭`: 심볼(변수, 메서드 등)의 선언부로 이동합니다.
7. `Cmd + /`: 현재 라인을 주석 처리합니다.
8. `Cmd + Shift + /`: 선택한 코드 블록을 주석 처리합니다.
9. `Cmd + Alt + T`: 선택한 코드에 if, for, try/catch 등을 감쌉니다.
10. `Cmd + F`: 현재 파일에서 검색합니다.
11. `Cmd + R`: 현재 파일에서 교체합니다.
12. `Cmd + Shift + F`: 전체 프로젝트에서 검색합니다.
13. `Cmd + Shift + R`: 전체 프로젝트에서 교체합니다.
14. `Cmd + E`: 최근 열었던 파일 목록을 보여줍니다.
15. `Cmd + Alt + ←/→`: 열었던 파일 사이를 앞뒤로 이동합니다.
16. `Ctrl + Space`: 기본 코드 자동완성을 활성화합니다.
17. `Ctrl + Shift + Space`: 스마트 코드 자동완성을 활성화합니다.
18. `Ctrl + Shift + Enter`: 현재 라인을 완성하고 새 라인을 시작합니다.
19. @@@ `Ctrl + P`: 메소드의 매개변수 정보를 보여줍니다. @@@@
20. `Ctrl + B`: 선언으로 이동합니다.
21. `Ctrl + Alt + B`: 구현으로 이동합니다.
22. `Ctrl + Shift + I`: 빠른 정의 보기를 제공합니다.
23. `Ctrl + /`: 라인 주석 처리/해제를 합니다.
24. `Ctrl + Shift + /`: 블록 주석 처리/해제를 합니다.
25. `Ctrl + Y`: 현재 라인을 삭제합니다.
26. `Ctrl + D`: 현재 라인을 복사하여 아래에 붙여넣습니다.
27. `Ctrl + Shift + J`: 아래 라인과 현재 라인을 합칩니다.
28. `Ctrl + Alt + L`: 코드 포맷팅을 진행합니다.
29. `Ctrl + Alt + O`: 불필요한 import를 정리합니다.
30. `Ctrl + R`: 파일 내에서 교체를 합니다.
31. `Alt + Enter`: 현재 컨텍스트에서 사용 가능한 액션을 보여줍니다 (예: 코드 수정 제안).
32. `Alt +` /`: 기본 코드 자동완성을 활성화합니다.
33. `Alt + Shift + Enter`: 현재 문장을 완성하고 새로운 라인으로 이동합니다.
34. `Alt + ↑/↓`: 현재 케어트를 이전/다음 메소드나 클래스로 이동합니다.
35. `Alt + ←/→`: 열린 탭 간에 전환합니다.
36. `Alt + F1`: 현재 파일이나 심볼을 다른 뷰로 전환합니다.
37. `Alt + F7`: 현재 커서의 요소 사용처를 찾습니다.
38. `Alt + F8`: 디버그 모드에서 표현식을 평가합니다.
39. `Alt + Insert`: 새 파일, 클래스, 테스트 등을 생성합니다.
40. `Alt + Home`: 네비게이션 바를 활성화합니다.
41. `Alt + 1`: 프로젝트 툴 윈도우를 열거나 닫습니다.
42. `Alt + 2`: 즐겨찾기 윈도우를 열거나 닫습니다.
43. `Alt + Shift + F10`: 실행/디버그 구성을 선택합니다.
44. `Alt + Shift + F9`: 디버그 모드에서 선택된 구성을 실행합니다.
45. `Alt + Shift + L`: 현재 파일에서 로컬 변수를 리팩터링합니다.
